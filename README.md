
# Credit Default Predictor — Full ML Pipeline

A complete machine learning project that predicts whether a customer will default on a loan, using a **Random Forest Classifier** and a modular, production-ready architecture.

## ✅ Features
- Full preprocessing pipeline
- Config-driven model training
- Model registry with version metadata
- Evaluation script
- Sample dataset included
- Easily replace with your own dataset
- Works locally or in cloud environments
